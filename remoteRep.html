Adding a Remote Repository

Purpose:

This allows users to open a repository quickly without having to clone it locally. It also allows users to take advantage of virtual file system and allows them to create pull requests in vs code

Step 1:

For this to work you must install remote repository extension

Step 2:

Open new repository through green remote indicator in the lower left screen

Step 3:

A menu will open once you switch into the extension. Scroll down and open remote repository
