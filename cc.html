Creating Repository

Step 1:

Go into your GitHub account and go to repositories. Click the "new" button, name the repository abd click the checkbox to add a README file. Then click "create repository" button.

Step 2:

Click the "<>code" dropdown button and copy the HTTPS link

Cloning Repository

Step 1:

Open VS Code back up and hit CTRL+Shift+P. Then search ">git: clone" and click on the searched item. Paste the HTTPS into the designated area, select where to place the file, and click the "open" button.

Step 2:

If the option pops up, click "yes, I trust the authors". Otherwise create a new file and call it "hello.txt"

Step 3:

Go to the source control panel and view the documents. New or changed files have a "U" next to the document title. Hit the + button to stage the document and the "U" will change to "A".

Step 4:

Type "my first commit" into the message line and click the "commit" button. Then click the expanded menu represented by "..." and click "push". Go back to GitHub, refresh your repository and check that the "hello.txt" file is there.
