Installing Git

Step 1:

In order to get started, we must install HomeBrew

Step 2:

Open your command center (Windows) or terminal (Mac) and paste the line of code provided on the HomeBrew site. Then hit enter. The system will then ask for your system's password. Enter it and HomeBrew will begin prompting you.

Step 3:

Copy, paster, and enter each line of code provided under "Next Steps". Once completed, enter in this line of code to check that you have Git.

Configure Git with VS Code

Step 1:

First, we must check that Git is enabled on our VS Code. To check this we must go to file > Preferences > Settings. Or if you're on a Mac you will go to Code > Settings... > Settings. Then type in Git:enabled into the search bar. Make sure the box next to it is checked.

Step 2:

Now, we must go to GitHub and create or log in to an account. Once that is completed we will then reopen the command prompt

Step 3:

Here we are going to configure our log in with the following code using the email and username created for your Git log in.

git config --global user.email "youremail@here.com" git config --global user.name "Example-Name"
