Creating a Pull Request

Purpose:

Pull requests allows you tell others about changes you've pushed to a branch in a repository on GitHub.

Step 1:

Make the necessary change(s) on your document(s).

Step 2:

Go to the source control panel and type a message into the message box.

Step 3:

Click the “create pull request” button next to the refresh button.

Step 4:

Choose a name for the pull request.

Step 5:

Create a branch name and hit enter.

Step 6:

A message will pop up saying a pull request was created and if you would like to switch to that branch to keep working

Step 7:

Switch to the new branch

GitHub Pull Request Extension

Once you have pull request extension, you can view the changes made and add comments to your files. You can also merge your pull request indirectly and delete the branch from this window.
